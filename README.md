# defaultuserconfigandlibs
A library for all of my other mods, this is most often required if my other mods have configs.

Readme from modrinth: https://modrinth.com/mod/defaultuserconfigandlibs

This jar file does nothing on its own, its a library for my other mods. Currently, it is also in alpha with an icon made in mspaint.

It can be used to detect the minecraft version, serve icons, detect fabric version, date and time, and host OS for debugging purposes, it is entirely local, dont believe me? aw

Required by my current and possibly future mods: WorldVersionBackport - Modrinth: https://modrinth.com/project/worldversionbackport
